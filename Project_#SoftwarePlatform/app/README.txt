ИСПОЛЬЗОВАТЬ RUN.BAT ДЛЯ ЗАПУСКА ПРОГРАММЫ!

Для запуска программы необходима вертуальная среда и Flask! Используйте батник run.bat для установки и запуска!
Батник установит виртуальную среду и Flask, необходимые для создания своего портфолия, но только в том случае, если виртуальная среда уже не установлена.
Если виртуальная среда установлена, то процесс сразу перейдет к установке Flask и запуску программы.

Для проверки работы батника (и программы соответственно) МОЖНО удалять папку venv!
